---
title: Duplicate Detection
description: How MakerVault identifies duplicate files across all your folders.
---

MakerVault automatically identifies duplicate files across all your folders, even if they have different filenames or are stored in different locations.

## How it works

1. When a file is scanned, MakerVault fingerprints its contents in the background.
2. Files with identical content are matched regardless of filename or location.
3. Duplicates are grouped together so you can review them.

## Viewing duplicates

Click **Duplicates** in the sidebar to see all duplicate groups. Each group shows:

- The number of copies found
- The file path for each copy
- Which watched folder each copy lives in

## Cleaning up duplicates

In the Duplicates view, you can select files you no longer need and click **Move to Trash** to clean them up. Files are moved to your system Trash (not permanently deleted), so you can recover them if needed. MakerVault shows you how much disk space you've reclaimed after cleanup.

## Ignoring duplicates

Some duplicates are intentional — you might keep the same file in multiple project folders on purpose. Click **Ignore** on any duplicate group to dismiss it from the list. Ignored groups won't appear in the Duplicates view, so you can focus on the duplicates that actually need attention.

If you change your mind, the summary bar shows how many groups you've ignored. Click the ignored count to restore all ignored groups back into the list.
