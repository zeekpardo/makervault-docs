Welcome to the MakerVault documentation. MakerVault is a local-first desktop file organizer built for makers who use laser cutters, CNC routers, vinyl cutters, and 3D printers.

**Find any design file in seconds. Tag it. Open it. Done.**

## What MakerVault does

MakerVault solves the universal pain point of having thousands of design files scattered across folders with no way to search, tag, or attach context to them. It handles:

- **File indexing** — Point MakerVault at your folders and it automatically scans and indexes 25+ file types including SVG, DXF, AI, STL, G-code, LightBurn, and more.
- **Instant search** — Full-text search across filenames, tags, notes, and source URLs. Deep search lets you search inside file contents.
- **Hierarchical tagging** — Create nested tags up to 3 levels deep (e.g., Holiday > Christmas > Ornaments) and filter by any level.
- **Spacebar preview** — Press Space to preview any file without opening another app. SVGs, images, PDFs, fonts, and LightBurn files all render natively.
- **LightBurn integration** — MakerVault reads inside LightBurn files to extract notes, cut settings, text elements, and font names.
- **Source tracking** — Record where files came from (Design Bundles, Etsy, personal designs) with auto-detection from URLs.
