---
title: Multi-Filter
description: Combine search text with file type, tag, source, favorites, and note filters.
---

MakerVault's multi-filter system lets you stack multiple criteria to narrow down your file list. All filters work together with AND logic — a file must match every active filter to appear.

## Available filters

| Filter | How it works |
|---|---|
| **Search text** | Matches filenames, tags, notes, source URLs |
| **File type** | Filter by category (Vector, Raster, 3D) or software (LightBurn, SVG) |
| **Tags** | Select one or more tags; parent tags include all descendants |
| **Sources** | Filter by file origin (Design Bundles, Etsy, etc.) |
| **Favorites** | Show only starred files |
| **Notes** | Show files with user notes, source notes, or search within notes |

## AND logic

Filters combine with AND logic. For example:

- **Tag: Christmas** + **Type: LightBurn** = Only LightBurn files tagged "Christmas"
- **Tag: Holiday** + **Tag: 2026** = Files tagged with both "Holiday" (or descendants) AND "2026" (or descendants)
- **Favorite** + **Source: Design Bundles** = Your favorited files from Design Bundles

## Active filter bar

When filters are active, a filter bar appears below the search bar showing each active filter as a removable chip. Click the **x** on any chip to remove that filter.

The search bar's filter icon shows a blue dot when any filters are active.

## Clearing filters

- Click the **x** on individual filter chips to remove them one at a time
- Click **All Files** in the sidebar to clear all filters and show everything
