---
title: Filter Popup
description: Access all filters from the search bar's filter icon.
---

The filter popup gives you access to all filtering options in one place. Click the filter icon inside the search bar to open it.

## Sections

### Favorites

Toggle the favorites filter to show only files you've starred.

### Notes

Two note types are available as separate filters:

- **My Notes** — Show files where you've added personal notes
- **Source Notes** — Show files with extracted metadata notes (e.g., LightBurn show notes)

When you check a note filter, a keyword search input appears. Type a keyword to search within the matching notes.

### Tags

A hierarchical tag tree with checkboxes. Checking a parent tag automatically includes all its descendants.

Each parent tag has an expand/collapse toggle, and there's an "expand all / collapse all" toggle at the top of the tag section.

Tags show their file counts next to the name.

### Sources

Multi-select checkboxes for all your sources. Each source shows the number of files associated with it.

### Hide Fonts

A **Hide Fonts** toggle lets you exclude font files (TTF, OTF, WOFF, WOFF2) from your file list. This is useful when you want to focus on design files without seeing hundreds of fonts mixed in. The toggle is in the Quick Filters section at the top of the filter popup.

### File Types

File types are grouped by category (Vector, Raster, G-code, 3D, Project, Document, Font) and show software names rather than raw extensions. Selecting "LightBurn" includes both `.lbrn` and `.lbrn2` files.

## Active filter indicator

When any filter is active, a blue dot appears on the filter icon in the search bar so you always know when your results are being filtered.
