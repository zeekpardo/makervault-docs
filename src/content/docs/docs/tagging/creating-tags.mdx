---
title: Creating Tags
description: How to create and apply tags to organize your design files.
---

Tags are the primary way to organize files in MakerVault. Unlike folders, a file can have multiple tags, and tags can be nested into hierarchies.

## Creating a tag

There are two ways to create tags:

### From the file detail panel

1. Select a file to open the detail panel on the right.
2. In the **Tags** section, start typing a tag name.
3. The autocomplete dropdown shows existing tags that match.
4. Select an existing tag, or type a new name and press Enter to create it.

### From the Tag Manager

1. Open **Settings** and go to the **Tags** tab.
2. Click **Add Tag**.
3. Enter a name, choose a color, and optionally select a parent tag.

## Applying tags

Select a file and use the tag input in the detail panel. Start typing and pick from the autocomplete suggestions.

Tags appear as colored chips on the file. Click the **x** on a chip to remove that tag from the file.

## Tag colors

Each tag has a customizable color that appears in the tag badge, sidebar tree, and filter chips. The default color is gray. You can change colors in the [Tag Manager](/tagging/tag-manager).

## Tag autocomplete

The autocomplete shows tags in hierarchical order. If you have `Holiday > Christmas`, typing "chr" shows "Holiday > Christmas" so you know exactly where it sits in the hierarchy.
