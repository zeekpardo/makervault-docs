---
title: Tag Manager
description: Create, edit, rename, recolor, and delete tags from Settings.
---

The Tag Manager in Settings gives you full control over your tag system.

## Accessing the Tag Manager

Open **Settings** (from the app menu or keyboard shortcut) and click the **Tags** tab.

## Creating a tag

1. Click **Add Tag**.
2. Enter a name.
3. Choose a color from the color swatches (a checkmark indicates the selected color).
4. Optionally select a parent tag from the dropdown to nest it under an existing tag.
5. Click **Save**.

## Editing a tag

Click the edit button on any tag in the list. You can change:

- **Name** — Renaming a tag updates it everywhere (file badges, sidebar, filters, search index).
- **Color** — Pick from the color swatch palette.
- **Parent** — Move a tag to a different position in the hierarchy. The parent dropdown shows tags at depth 0 and 1 only (enforcing the 3-level maximum).

## Deleting a tag

Click the delete button on a tag. Deleting a parent tag also deletes all its children (cascade delete). The tag is removed from all files that had it.

## Tag display

The Tag Manager shows each tag with:

- Color dot
- Full hierarchical path (e.g., "Holiday > Christmas > Ornaments")
- Number of files using that tag
