title: Managing Sources
description: Create, edit, and delete sources in Settings.
---

The Sources pane in Settings gives you full control over your source list.

## Accessing source management

Open **Settings** and click the **Sources** tab.

## Creating a source

1. Click **Add Source**.
2. Enter a name (e.g., "Design Bundles", "My Designs", "Client Work").
3. Optionally enter a URL for the marketplace.
4. Click **Save**.

Sources are also created automatically when you paste a purchase URL that MakerVault hasn't seen before. See [Auto-Detection](/sources/auto-detection).

## Editing a source

Click the edit button on any source in the list. You can change:

- **Name** — Displayed in the sidebar, filter popup, and file detail panel
- **URL** — The marketplace URL (shown as an external link)

## Deleting a source

Click the delete button on a source. Deleting a source removes the source association from all files that had it, but doesn't delete the files themselves. Purchase URLs and bundle names on individual files are preserved.

## Source list

The Sources settings table shows:

| Column | Description |
|---|---|
| **Name** | Source name |
| **URL** | External link to marketplace |
| **Files** | Number of files associated with this source |
| **Created** | When the source was created |

## Export/Import

Sources are included in [library exports](/data-management/export-import). When importing, sources are matched by UUID for merge, so renamed sources are handled correctly.
