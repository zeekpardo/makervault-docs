title: Tracking File Origins
description: Record where your design files came from — Design Bundles, Etsy, personal designs, and more.
---

MakerVault lets you track the source of every file in your library. This is especially useful when you buy bundles from multiple marketplaces and need to remember where something came from.

## What you can track

For each file, you can record:

- **Source** — The marketplace or origin (Design Bundles, Creative Fabrica, Etsy, personal designs, etc.)
- **Purchase URL** — The direct link to the product page
- **Bundle Name** — The specific bundle or product name the file came from

## Setting a source

There are three ways to assign a source to files:

- **Detail panel** — Select a file, then pick a source from the dropdown in the **Source** section. You can also paste a purchase URL and enter a bundle name.
- **Inline button** — Click the **+** button on any file row and choose **Source** to quickly assign one without opening the detail panel.
- **Bulk assignment** — Select multiple files, then click **Add** in the bulk bar and choose **Source** to apply the same source to all of them at once. See [Bulk Tagging & Sources](/tagging/bulk-tagging) for details.

## Sources in the sidebar

Sources appear in the left sidebar with file counts. Click a source name to filter your library to only files from that source.

## Sources in search

Source URLs and bundle names are included in the search index. Searching for "design bundles" finds all files with that in their source information.

## Filtering by source

Use the [filter popup](/search-filtering/filter-popup) to select one or more sources. Source filters combine with other filters using AND logic.
