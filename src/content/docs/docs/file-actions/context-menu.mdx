title: Context Menu
description: Right-click any file for quick access to all actions.
---

Right-clicking a file opens a context menu with quick access to all file actions.

## Available actions

| Action | Description |
|---|---|
| **Open** | Open in default application |
| **Open With** | Choose a specific application |
| **Preview** | Open Spacebar preview |
| **Reveal in Finder** | Show the file in Finder |
| **Copy File Path** | Copy full path to clipboard |
| **Rename** | Rename the file on disk |
| **Tags** | Quick access to add/remove tags |
| **Favorite** | Toggle the favorite star |

## Multi-select context menu

When multiple files are selected, the context menu adapts to show bulk actions. For example, you can tag multiple files at once from the context menu.

## Keyboard alternative

Most context menu actions have keyboard shortcuts:

| Shortcut | Action |
|---|---|
| **Space** | Preview |
| **Enter** | Open in default app |
| **Cmd+Shift+F** | Reveal in Finder |

## Renaming files

Selecting **Rename** from the context menu opens a dialog where you can type a new filename. The filename (without extension) is automatically selected for easy editing. MakerVault validates that the new name doesn't contain path separators and doesn't conflict with an existing file in the same folder. The file is renamed on disk and the database is updated to match.
