The Spacebar preview supports keyboard navigation so you can quickly scan through your files.

## Preview shortcuts

| Key | Action |
|---|---|
| **Space** | Open preview (when file selected) / Close preview (when open) |
| **Escape** | Close preview |
| **Left Arrow** | Previous file in the list |
| **Right Arrow** | Next file in the list |

## Workflow

A typical preview workflow:

1. Search or filter to narrow your file list.
2. Click the first file to select it.
3. Press **Space** to open the preview.
4. Use **Left/Right** arrows to scan through files.
5. When you find what you need, press **Space** to close, then double-click to open.

This lets you review many files quickly without leaving MakerVault or opening each file individually.
