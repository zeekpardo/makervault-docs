MakerVault's quick preview works like macOS Quick Look. Select a file and press **Space** to open a full-screen preview overlay.

## How to use it

1. Click a file to select it.
2. Press **Space** to open the preview.
3. Press **Space** or **Escape** to close it.

## What you see

The preview overlay shows:

- **File preview** — The file rendered natively (for supported types) or a placeholder icon
- **Metadata sidebar** — Filename, size, type, tags, notes, and embedded metadata
- **Open button** — "Open in [App]" to launch the file in its default application

## Inline preview

Previewable files also show a smaller inline preview directly in the file detail panel on the right sidebar, so you can see a preview without going full-screen.

## Supported types

See [Supported Previews](/preview/supported-previews) for the full list of what renders natively vs. what shows a placeholder.
