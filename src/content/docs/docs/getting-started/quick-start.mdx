This guide walks you through the core MakerVault workflow in about 2 minutes.

## 1. Add a folder

Click **Add Folder** in the sidebar and select the folder where you keep your design files. MakerVault scans it instantly.

## 2. Browse your files

Your files appear in a scrollable list showing the filename, file type label (e.g., "LightBurn", "SVG"), size, and date. Click any file to see its details in the right panel.

## 3. Search

Start typing in the search bar to find files by name, tags, or notes. Results appear as you type.

To search inside file contents (like LightBurn text elements or layer names), prefix your query with `/`:

```
/HOME
```

This finds any file containing "HOME" in its embedded content.

## 4. Tag your files

Select one or more files (Cmd+click for multiple) and use the tag input to apply tags. Tags can be hierarchical — for example, `Holiday > Christmas > Ornaments`.

Click a tag in the sidebar to filter your file list to only files with that tag.

## 5. Preview with Spacebar

Select a file and press **Space** to open a full-screen preview. SVGs, images, PDFs, and fonts render natively. LightBurn files show their embedded thumbnail and cut settings.

Use **Left/Right** arrow keys to navigate between files. Press **Space** or **Escape** to close.

## 6. Open in your app

Double-click any file to open it in its default application (LightBurn, Illustrator, etc.). Right-click for more options like **Reveal in Finder** or **Copy File Path**.
